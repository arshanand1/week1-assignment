List of Tags

  1. h1: Writes down the heading at Level 1.
  2. a: Used to define hyperlink which is used to link one webpage to another
  3. p: It defines a paragraph in HTML
  4. div: Defines a different section on the HTML page
  5. table: Defines a table in HTML
  6. ol: It defines ordered list
  7. b: It is used for making the text bold
